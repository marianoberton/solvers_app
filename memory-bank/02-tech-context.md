# Tech Context

## Stack Tecnológico
- **Frontend**: React 19.
- **Lenguaje**: TypeScript.
- **Build Tool**: Vite.
- **Estilos**: Tailwind CSS.
- **Iconos**: Lucide React.
- **Routing**: React Router DOM 7.

## Entorno de Desarrollo
- **Requisitos**: Node.js instalado.
- **Instalación**: `npm install`.
- **Ejecución Local**: `npm run dev` (Puerto default: 3000).
- **Build**: `npm run build`.

## Configuración
- `vite.config.ts`: Configuración base de Vite, define alias `@` y variables de entorno.
- `tailwind.config` (en `index.html` script): Configuración de temas y colores (Slate palette).

## Scripts
- `dev`: Inicia servidor de desarrollo.
- `build`: Compila para producción.
- `preview`: Vista previa del build.
